# Feature Spec: Unum People Tools Section

## Overview
Criação de uma seção dedicada às ferramentas proprietárias da Unum People no site institucional, detalhando o hub de produtividade que inclui recursos gratuitos e soluções avançadas.

## Requirements

### R1: Content & Value Proposition
- [ ] **R1.1**: Apresentar o hub `tools.unumpeople.com.br` como uma central de produtividade com recursos gratuitos.
- [ ] **R1.2**: Listar sub-ferramentas (Análise de Portfólio, Simulador de Precificação, Viabilidade de Campanhas).
- [ ] **R1.3**: Reforçar o benefício de "Inteligência de Dados" e "Agilidade".

### R2: Visual Design
- [ ] **R2.1**: Seguir a estética ultra-minimalista e vanguardista do projeto.
- [ ] **R2.2**: Utilizar componentes de card com ícones modernos.
- [ ] **R2.3**: Efeito visual de "Hub" ou "Central de Comando".

### R3: Interactivity
- [ ] **R3.1**: Botão de CTA direcionando para o subdomínio de ferramentas.
- [ ] **R3.2**: Animações suaves via Framer Motion ao rolar.

## Constraints
- Manter consistência com a paleta de cores Unum People.
- Usar `next/image` e Lucide React para ícones.
- Código idiomático no `src/app/page.tsx`.
